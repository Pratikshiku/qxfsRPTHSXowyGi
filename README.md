## Supermarket Management System with Mobile QR Code Inbound and Outbound

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/2c3a28f8d8f64e52bfd437408efde4a2/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): System Administrator, Sales Staff, Warehouse Manager.

**System Administrator Functions:**
1. Login
2. Sales Dashboard
3. Sales Records
4. Points Exchange Records
5. Department Management
6. Employee Management
7. Password Change
8. Personal Information Edit
9. Warehouse Management
10. Outbound Details
11. Inbound Details
12. Supplier Information
13. Inventory Statistics
14. Member Information Management
15. System Role Management
16. System Menu Management
17. Product Category Management
18. Product Information Management
19. Points Product Management
20. Sales Statistics
21. Secure Logout


**Sales Staff Functions:**
1. Login
2. Member Information Management
3. Personal Information Edit
4. Password Change
5. Sales Dashboard
6. Sales Records
7. Points Exchange Records
8. Secure Logout

**Warehouse Manager Functions:**
1. Login
2. Product Sales Statistics
3. Product Information Management
4. Points Product Management
5. Product Category Management
6. Personal Information Edit
7. Change Password
8. Warehouse Management
9. Outbound Details
10. Inventory Statistics
11. Supplier Information
12. Inbound Details
13. Secure Logout
